## 功能特性

- **大模型常识**：产业视角、工程视角、OpenClaw 层级架构及配置目录说明
- **常见问题**：20+ 个真实问题解答，覆盖安装、网络、Gateway、频道等
- **命令大全**：20+ 常用命令，涵盖诊断、配置、频道、模型、Skill 管理
- **优化建议**：知识库组织规范（soul.md/memory.md/agents.md）及性能调优

## 安装

```bash
npx clawhub install openclaw-newbie-faq
```

## 使用

安装完成后，对 OpenClaw 说"启动新手帮助"启动 Web 服务。

访问地址：http://localhost:34567

## 启动 Skill

安装后请对 OpenClaw 说"**新手帮助**"或"**我是新手**"，即可自动启动 34567 端口的 Web 服务。
